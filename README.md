# Hotel Booking Cancellation Risk Model (Excel Project)

This project analyzes hotel booking data and builds a predictive **cancellation-risk scoring model** entirely in Excel.  
It includes interactive dashboards, feature-engineered risk scoring logic, validation checks, and scenario analysis tools.  
The project is currently in progress and will continue to expand.
